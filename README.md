# Breast-Cancer-Analysis-METABRIC
"An end-to-end Machine Learning project to predict breast cancer survival outcomes using the METABRIC dataset."
Project Title: METABRIC Breast Cancer Survival Prediction

Algorithm: Random Forest Classifier

Accuracy: 81%

Key Features: Relapse Free Status, Age, NPI.

Techniques: Data Cleaning, SMOTE (Class Balancing), SHAP (Explainable AI), and Model Persistence.
